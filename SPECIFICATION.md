# Student Grade Management System

## 1. Introduction

### Project Title

Student Grade Management System

### Domain

Education

Educational institutions need reliable systems to manage student academic records. Lecturers often handle many students and subjects, making it difficult to manage grades using manual methods such as spreadsheets or paper-based records.

### Problem Statement

Managing student grades manually can lead to errors, loss of information, and inefficient record keeping. Lecturers require a reliable system that can store, manage, and retrieve student grade information easily.

### System Purpose

The purpose of the Student Grade Management System is to provide a centralized platform for recording and managing student academic grades. The system allows lecturers to store student information, record subject marks, and view grade records efficiently.

### Individual Scope

This project focuses on designing a simple system that demonstrates how student grade information can be managed electronically. The system will include functionality for adding students, recording grades, and viewing academic results.

The project will demonstrate system specification and architectural design using appropriate documentation and diagrams.
