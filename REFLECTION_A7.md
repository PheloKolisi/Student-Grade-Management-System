# Reflection – Assignment 7

One of the main challenges in this assignment was selecting the most appropriate GitHub project template. Each template has different features, and it was necessary to evaluate which one best supports Agile development.

Another challenge was customizing the Kanban board. Adding new columns such as "Testing" and "Blocked" required understanding how workflows operate in real-world systems.

Linking user stories from previous assignments to GitHub Issues was also challenging, as it required ensuring consistency and traceability between assignments.

Comparing GitHub Projects to other tools such as Trello and Jira highlighted that while GitHub is powerful for developers, it may require more setup compared to simpler tools like Trello.

Overall, this assignment improved my understanding of Agile project management and how Kanban boards are used to track development tasks effectively.
