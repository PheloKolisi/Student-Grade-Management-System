# Sprint Plan – Student Grade Management System

## Sprint Goal

Develop the core functionality of the system, including adding students, adding subjects, capturing grades, and viewing results.

---

## Selected User Stories

* US-001 Add student
* US-002 Add subject
* US-003 Capture grades
* US-005 View grades
